<h1>Tic-Tac-Toe</h1>
A browser-based Tic-Tac-Toe game where two players compete and try to be the first to get three of the same symbol in a row.
<h1>Languages Used</h1>
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
